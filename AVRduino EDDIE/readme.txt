 AVRduino EDDIE is bussiness sized microcontroller board based around the AVR128DB48..
Features include, UPDI header, selectable USB serial, jumper for 3.3 or 5 volts on PORTC, 
footprint for half sized external clock for speeds upto 40 mHz, and foot print for 24c256
memory chip..Input power from either a standard arduino 9 vdc power supply, the usb serial
port via a jumper or the 5 volt pin on the power haeder..
